# notion
일단은 내가 쓸려고 만든거. 노션 입력, 쓰기 등 하기

노션 api 버전 2025-09-03 버전으로 변경

>**데이터베이스**<br>
>1. 데이터베이스
>    - 쓰기: 페이지 생성(데이터소스가 둘 이상이면 작동 안함)
>    - 읽기: 데이터 소스 가져오기
>2. 데이터 소스
>    - 읽기: 페이지 정보 가져오기
>3. 페이지
>    - 업데이트: 페이지 정보 수정
>    - 지우기: 페이지 지우기

>**페이지**<br>
>곧 만들기

>**블럭**<br>
>곧 만들기

## 데이터베이스

| 제목 |    날짜    |  가격 | 
|------|------------|-------|
| 피자 | 2020-10-10 | 15000 |

```python
api_key = "<<api_key>>"
database_id = "<<database_id>>"
notion = NotionDatabase(api_key, database_id)

# 데이터소스 가져오기
notion.read()
# 데이터소스 리스트
notion.value
# 페이지 쓰기
notion..write(
    NotionObjec()
    .title("제목", "치킨")
    .number("가격", 10000)
    .date("날짜", "2025-09-03")
)
```

| 제목 |    날짜    |  가격 | 
|------|------------|-------|
| 피자 | 2020-10-10 | 15000 |
| 치킨 | 2025-09-03 | 10000 |


```python
data_source = notion.value[0]
# 페이지 가져오기
data_source.read()
# 페이지 가져오기
data_source.value
```
```python
page = data_source.value[0]
# 페이지 업데이트
page.update(
    NotionObjec()
    .title("제목", "치킨반마리")
    .number("가격", 5000)
)
```
|    제목    |    날짜    |  가격 | 
|------------|------------|-------|
|    피자    | 2020-10-10 | 15000 |
| 치킨반마리 | 2025-09-03 |  5000 |


```
# 페이지 삭제
page.remove()
```
